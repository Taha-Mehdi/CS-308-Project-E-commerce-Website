Price displayed as NaN when backend sends string

**Area:** Cart Page  
**Severity:** Low  
**Status:** Fixed

---

## Summary
Some products display NaN as price or subtotal when the backend sends the price as a string, for example `"199.99"`.

---

## Steps to Reproduce
1. Add a product where API returns `"price": "199.99"`.
2. Open cart page.
3. Observe price and subtotal.

---

## Expected Result
Price renders correctly and subtotal is numeric.

---

## Actual Result
Price or subtotal displays as `NaN`.

---

## Root Cause
String values were used directly in arithmetic
